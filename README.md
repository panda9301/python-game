#  人生模擬器 - Life Simulator

一款互動式網頁遊戲，讓你在 30 天內體驗人生的無限可能。沒有正確答案，只看你想過怎樣的人生。

## 遊戲概述

這是一個簡單卻深刻的人生模擬遊戲。你將扮演主角，在 30 天內通過每日選擇來塑造自己的人生軌跡。每個選擇都會影響你的四大屬性：金錢、學習力、心情和健康。

### 四大核心屬性
- ** 金錢** - 影響生活品質，可通過打工獲得
- ** 學習力** - 代表知識積累，通過讀書提升
- ** 心情** - 生活滿足度，受各項活動影響
- ** 健康** - 身體狀況，過度勞累會下降

##  遊戲玩法

### 每日四項選擇
1. ** 認真讀書** - 提升學習力，但損耗心情和健康
2. ** 去打工** - 賺取金錢，但會感到疲勞
3. ** 好好休息** - 恢復心情和健康
4. ** 出去玩** - 大幅提升心情，花費金錢

### 隨機事件
遊戲中隨機觸發各種生活事件：
-  中獎 - 意外之財
-  感冒 - 健康下降
-  朋友揪出去玩 - 心情提升，花費金錢
-  靈感爆發 - 學習力大幅提升

### 遊戲結局
根據你的最終屬性值，遊戲會給出 6 種不同的結局：

| 結局 | 條件 |
|------|------|
|  身體累壞了 | 健康 ≤ 20 |
|  人生勝利組 | 學習力 ≥ 80 且 心情 ≥ 60 |
|  小小財富自由 | 金錢 ≥ 2000 且 心情 ≥ 40 |
|  高分低樂 | 學習力 ≥ 80 且 心情 < 40 |
|  開心就好派 | 心情 ≥ 80 |
|  平凡人生 | 其他情況 |

##  技術棧

- **後端** - Python Flask
- **前端** - HTML5 + CSS3
- **模板引擎** - Jinja2

##  系統需求

- Python 3.7+
- Flask
- 現代網頁瀏覽器

## 🚀 安裝與執行

### 1. 安裝依賴
```bash
pip install flask
```

### 2. 執行應用
```bash
python life_sim.py
```

### 3. 進入遊戲
在瀏覽器中打開： `http://127.0.0.1:5000`

##  遊戲策略 Tips

- **平衡很重要** - 不要過度追求單一屬性
- **健康最優先** - 健康歸零遊戲提早結束
- **隨機事件** - 每天有 30% 機率觸發隨機事件
- **多次遊玩** - 每次遊戲都會有不同結果
